
### Installation

```markdown
## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/elithaxxor/mac_tool_installer.git
   cd mac_tool_installer/CLI
   ```

2. Make the installer script executable:

   ```bash
   chmod +x comprehensive_installer.sh
   ```

3. Run the installer script:

   ```bash
   ./comprehensive_installer.sh
   ```

4. Follow the on-screen prompts to install the desired tools.
```

### Usage

```markdown
## Usage

The `comprehensive_installer.sh` script provides a menu-driven interface to install tools individually or all at once.

### Main Menu Options:
1. Install all tools at once
2. Install tools individually
3. Get information about tools
4. Exit

### Example:
To install all tools at once, select option 1 and confirm the installation.
```
